# LineAnalyzer
* python勉強がてらに作って見ましたー
* iphoneのバックアップファイルからlineのトークcsvを生成します
* Python 2.7.13で動作確認

## result files
* `int(5).csv`
     * トークルームID別ファイルが作成され、トーク内容がutf8で入ります
     * macのように絵文字対応のterminalがある場合は絵文字も見れます
* zchat.csv
     * トークの出力された順番です。新しいトークがどこなのかどうか悩む時はこれをみるとわかります
     * `cat zchat.csv | uniq -c `とかしてください
